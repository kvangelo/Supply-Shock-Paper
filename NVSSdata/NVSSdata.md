This folder contains detailed synthetic mortality trends in the United States.

The folder clustercharts contains detailed charts pertaining to the cluster analysis referenced in the appendix. 
Here, state trends were matched based on 2022-2023 trends. Based on the cluster analysis, it appears the downturn first occured on the East coast. 

The folder codrugcharts contains 12-month smoothed mortality rates pertaining to drug deaths. 
For each multiple cause of death alcohol related deaths, heroin related deaths, cocaine related, and psychostimulant related deaths we plotted syntehtic opioid codrug mortality. 
For example, the heroin codrug chart shows the trend in heroin related deaths with syntethic opioids also listed as a cause of death, as well as the trend without syntehtic opioid present.

The folder statetrends contains two-month smoothed mortality rates underlying the cluster analysis referenced in the appendix.
